# 🧹 Smart CSV & Excel Cleaner

A Python tool that automatically cleans messy CSV/Excel files and generates a visual report.

## 🔧 What it does
- Removes duplicate rows
- Removes empty/blank rows
- Cleans column names (lowercase, no spaces)
- Strips extra whitespace from data
- Generates a visual report with charts

## 📊 Output
- Cleaned CSV file
- PNG report showing missing values + rows kept vs removed

## 🚀 How to run
```bash
pip install pandas matplotlib openpyxl
python cleaner.py
```

## 💼 Use case
Perfect for small businesses, data entry teams, and anyone with messy Excel data.

## 🛠️ Built with
- Python, Pandas, Matplotlib
